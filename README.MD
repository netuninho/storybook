# Estudo sobre Storybook utilizando Nextjs | React | Typescript | Yarn

## Passo a passo do projeto:

- `yarn init -y` 
- `yarn add react react-dom next`
- `yarn add --dev typescript @types/react @types/node`
- `npx gitignore node`
- `npx storybook init`
